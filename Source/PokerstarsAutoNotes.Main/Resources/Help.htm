Open and Save the Pokerstars Notes File

Open Notes File

Go to the file menu select open notes and move to the directory where your Pokerstars notesfile is located then click open. The Pokerstars notesfile location can be seen in the Pokerstars software. Go to the options menu and select player notes, than click file location.
Back to Top

Save Notes File

Go to the file menu select save notes. The existing file will be overwritten.
Back to Top

Save Notes File As

Go to the file menu select save notes as. You will be asked for a filename and the location where to save the file. Make sure you have the proper rights to save the file in the selected directory.
Back to Top

Autolabel

In the menu players click autolabel to change the labels of the players automatically.
For example look at the following table, all players with a vpip greater or equal 15 and less than 30
assigned to custom label 3, all players with a vpip greater or equal 30 assigned to custom label 4,
players with more than 5k hands and more than -0,5 bb/100 assigned to custom label 5,
players with with more than 50k hands and more than 1 bb/100 assigned to custom label 6.
All combinations are possible. Note: if a label does not have any values in the 4 boxes
the label will not be used. if not all values of a label are empty the values for not filled
ratings will be for VPip = 0, PFR = 0, BB/100 = -100000, Hands = 0.
In the example if a player has a Vpip of 30 or more but also more than 50k hands and bb/100 > 1
he will be assigned to custom label 4. The autolabeling takes always the highest Vpip 1st
than the PFR and bb/100 and at last the hands.
